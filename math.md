# Math Functions Reference

| Function | Description | Syntax | Arguments | Example |
|----------|------------|--------|-----------|---------|
| acos | Возвращает арккосинус значения (в радианах). | `acos(value)` | value — косинус угла в радианах | `acos(0.5) → 1.0471975511966` |
| asin | Возвращает арксинус значения (в радианах). | `asin(value)` | value — синус угла в радианах | `asin(1.0) → 1.5707963267949` |
| atan | Возвращает арктангенс значения (в радианах). | `atan(value)` | value — тангенс угла в радианах | `atan(0.5) → 0.463647609000806` |
| atan2 | Возвращает арктангенс dy/dx с учётом квадранта. | `atan2(dy, dx)` | dy — разница по оси Y<br>dx — разница по оси X | `atan2(1.0, 1.732) → 0.523611477769969` |
| ceil | Округляет число вверх. | `ceil(value)` | value — число | `ceil(4.9) → 5` |
| clamp | Ограничивает значение заданным диапазоном. | `clamp(min, input, max)` | min — нижняя граница<br>input — значение<br>max — верхняя граница | `clamp(1,5,10) → 5` |
| cos | Возвращает косинус угла. | `cos(angle)` | angle — угол в радианах | `cos(1.571) → 0.0007963` |
| degrees | Преобразует радианы в градусы. | `degrees(radians)` | radians — число | `degrees(π) → 180` |
| exp | Возвращает экспоненту числа. | `exp(value)` | value — число | `exp(1) → 2.71828` |
| floor | Округляет число вниз. | `floor(value)` | value — число | `floor(4.9) → 4` |
| ln | Возвращает натуральный логарифм. | `ln(value)` | value — число | `ln(1) → 0` |
| log | Возвращает логарифм по заданному основанию. | `log(base, value)` | base — число<br>value — число | `log(2,32) → 5` |
| log10 | Возвращает десятичный логарифм. | `log10(value)` | value — число | `log10(100) → 2` |
| max | Возвращает наибольшее значение. | `max(v1, v2, …)` | values — числа | `max(2,10.2,5.5) → 10.2` |
| min | Возвращает наименьшее значение. | `min(v1, v2, …)` | values — числа | `min(20.5,10,6.2) → 6.2` |
| pi | Возвращает число π. | `pi()` | — | `pi() → 3.14159` |
| radians | Преобразует градусы в радианы. | `radians(deg)` | deg — число | `radians(180) → 3.14159` |
| rand | Возвращает случайное целое число в диапазоне. | `rand(min,max[,seed])` | min — целое<br>max — целое<br>seed — опционально | `rand(1,10) → 8` |
| randf | Возвращает случайное число с плавающей точкой. | `randf([min,max,seed])` | min/max — числа<br>seed — опционально | `randf(1,10) → 4.59` |
| round | Округляет число до указанного количества знаков. | `round(value[,places])` | value — число<br>places — опционально | `round(1234.567,2) → 1234.57` |
| scale_exponential | Преобразует значение с использованием экспоненциальной кривой. | `scale_exp(v,dmin,dmax,rmin,rmax,e)` | диапазон входа и выхода + экспонента | `scale_exp(5,0,10,0,100,2) → 25` |
| scale_linear | Линейно масштабирует значение. | `scale_linear(v,dmin,dmax,rmin,rmax)` | диапазон входа и выхода | `scale_linear(5,0,10,0,100) → 50` |
| scale_polynomial | Преобразует значение с использованием полиномиальной кривой. | `scale_polynomial(v,dmin,dmax,rmin,rmax,e)` | диапазон входа и выхода + степень | `scale_polynomial(5,0,10,0,100,2) → 25` |
| sin | Возвращает синус угла. | `sin(angle)` | angle — радианы | `sin(1.571) → 0.999999` |
| sqrt | Возвращает квадратный корень. | `sqrt(value)` | value — число | `sqrt(9) → 3` |
| tan | Возвращает тангенс угла. | `tan(angle)` | angle — радианы | `tan(1.0) → 1.5574` |
